# Skills Reference

## Available skills

- **customize-opencode** — Use when editing opencode's own configuration files.
- **find-skills** — Use when the user asks how to do something that might be available as an installable skill.

## Agent types

- **explore** — Fast agent for codebase exploration and searching. Use for finding files, searching patterns, and answering codebase questions.
- **general** — General-purpose agent for complex multi-step tasks.
